# Catálogo de Productos

## Descripción

Aplicación web desarrollada utilizando HTML5, CSS3 y JavaScript
Permite visualizar un catálogo de productos consumiendo una API REST, filtrar por categorías y administrar un carrito de compras con persistencia local.

Los productos se obtienen desde:

https://dummyjson.com/products


## Características principales

- Catálogo generado dinámicamente.
- Barra lateral filtros por categoría.
- Visualización de productos generados dinámicamente.
- Persistencia de datos utilizando LocalStorage.
- Manejo de estados de carga y error.
- Diseño Flexbox y CSS Grid.
